# 此git為自主練習使用ROS2 turtlesim 實現 PID Control

## P_control 是讓 烏龜 到 指定點，實現線性 PID Control

## Rotate_p_control 是讓烏龜 到 指定角度，實現角度 PID Contorl

## follow_control 是結合前兩者，並且使烏龜可以畫出六邊形
